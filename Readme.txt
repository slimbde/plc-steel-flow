# PLC STEEL FLOW

Вычитывает из контроллеров скорость потока из промковша в ТПА и передает в ИТЦ

Внимание!
В этом решениии внесена микромодификация обработчика S7:
- убран делегат получения данных
- метод `Read` теперь сам возвращает данные
- сделано потому, что необходимо вычитывать обработчики синхронно, идентифицируя их по IP
- по идее, судя по конфигу, мы обрабатываем только один ТПА, можно было ничего и не менять
- но с возможным заделом на будущее, их число может быть увеличено
